# 📊 Sales Data Analysis

## Overview
This project analyzes sales data using Python to uncover business insights through data cleaning, visualization, and exploratory data analysis.

## Objectives
- Clean and preprocess sales data
- Analyze monthly sales trends
- Identify top-selling products
- Find the best time to advertise
- Discover products frequently bought together
- Analyze weekly sales patterns

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Structure

Sales-Data-Analysis/
│── Sales_Data_Analysis.ipynb
│── Sales Data.csv
│── README.md

## Key Insights
- Monthly sales trends
- Product performance
- Customer purchasing behavior
- Sales by weekday
- Business recommendations
